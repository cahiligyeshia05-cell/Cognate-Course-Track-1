This Laboratory Exercise (1.2) expanded upon the previous activity. The only difference is the software aspect. The hardware remains the same.

In terms of software, the exercise specifically required for the LED lights to light up from the outermost to the innermost lights. The code utilized for loops and an array containing a specified order of values which corresponded to the pin numbers. The loops employ the process of looping through the specific array order, arranged in pairs starting with the first pair (leftmost and rightmost as the outermost pair, ex.: pin 13 and 2, and so on).
